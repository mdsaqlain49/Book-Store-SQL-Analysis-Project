# 📚 Book Store SQL Analysis Project

## 📌 Project Overview
This project demonstrates SQL skills by building and analyzing a bookstore database using PostgreSQL. The database contains information about books, customers, and orders. SQL queries were used to solve business-related problems such as revenue analysis, customer behavior, inventory tracking, and sales performance.

## 🎯 Objectives
- Create a relational database for a bookstore
- Import CSV datasets into PostgreSQL
- Perform data analysis using SQL queries
- Generate business insights from customer and sales data

## 🗂 Database Structure
The project contains three tables:

### Books Table
Stores information about books including:
- Book ID
- Title
- Author
- Genre
- Published Year
- Price
- Stock

### Customers Table
Stores customer details including:
- Customer ID
- Name
- Email
- Phone
- City
- Country

### Orders Table
Stores order information including:
- Order ID
- Customer ID
- Book ID
- Order Date
- Quantity
- Total Amount

## 📊 Business Problems Solved
This project answers questions such as:

- Retrieve books by genre
- Find books published after a specific year
- Calculate total revenue generated
- Identify best-selling books
- Analyze customer purchase behavior
- Find highest spending customers
- Track inventory and remaining stock
- Determine genre-wise book sales
